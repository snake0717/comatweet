##　アプリケーション名
コマツイート

## アプリケーション概要
漫画のコマ投稿ができるアプリケーション。投稿された画像に対してはコメント、いいねができる。
気に入ったユーザーがいた場合フォローできる。

## 利用方法
新規登録、もしくはテスト用アカウントでログイン後、トップページの右上にあるコマ投稿ボタンから画像を投稿する。
